FPA_FOD_20210617.accdb \Data Microsoft Access database containing a spatial database of 1992-2018 wildfires in the United States. (FPA=Fire Program Analysis, FOD=Fire Occurrence Database). NOTE: this database is available in 4 different formats (accdb, gdb, gpkg, and sqlite), which are each a separate download.
FPA_FOD_20210617.gdb \Data ESRI ArcGIS geodatabase containing a spatial database of 1992-2018 wildfires in the United States. This geodatabase includes 2 feature classes: Fires and NWCG_UnitIDActive which is the same tabular data found in the Access database above, but also includes the point locations of the wildfires for ready use in ArcGIS. (FPA=Fire Program Analysis, FOD=Fire Occurrence Database). NOTE: this database is available in 4 different formats (accdb, gdb, gpkg, and sqlite), which are each a separate download.
FPA_FOD_20210617.gpkg \Data OGC GeoPackage open source data format implemented as a SQLite database containing a spatial database of 1992-2018 wildfires in the United States. (FPA=Fire Program Analysis, FOD=Fire Occurrence Database). NOTE: this database is available in 4 different formats (accdb, gdb, gpkg, and sqlite), which are each a separate download.
